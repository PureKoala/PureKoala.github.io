What’s Machine Learning

定义很多，我更喜欢Tom Mitchell的：

A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E.

大体上，机器学习的问题可以被划分为两种：Supervised learning 和 Unsupervised learning

Supervised learning

Supervised learning 是一类已知数据和其对应输出结果的机器学习类型，我们的任务就是探索输入和输出之间究竟存在着怎样的关系；对于不同的问题，Supervised learning 可以被分为回归（Regession）和分类（Classification）：

Regrssion

在回归问题中，我们需要通过不连续的数据得到输入与输出之间连续的关系，也就是对数据进行函数拟合，并且拟合的函数曲线可以用于对输入的预测。

就比如，通过未发售手机已公布的参数进行对该手机售价的预测，虽然近几年品牌溢价日趋明显，华为P40居然要卖4488就离谱，这是难以预测的，但是我们仍可以通过合理的拟合将误差控制在一个较小的范围内。Regression这种对于结果的预测可以广泛地应用于多种场景之中，对其的学习是学习机器学习过程中很重要的一环。

Classification

在分类问题中，相反，我们需要将数据划分入不同的分组之中，例如，给定一个有肿瘤病人的各项数据，去判断他的肿瘤是良性的还是恶性的。

Unsupervised learning

在Unsupervised learning中，我们几乎或者根本不了解结果应该是什么样的，我们可以通过机器学习将这些数据进行分组，而不必知道变量的影响；其与上述Classification之间的区别在于，Classification是已知这些分组的含义以及所需分组的数量，而Unsupervised learning的已知条件只有这些数据的特征，不知道每个分组有着怎样的意义，甚至不知道该分为几组，以下为上述两种分类的简单图示：

Cost Function

Cost Function是用于评估我们假设函数的准确性的一个指标，就实际而言，它表征的是每个数据的预测结果与实际结果之间的差值的平方的平均值，公式长这样：
$$
J\left(\theta_{0}, \theta_{1}\right)=\frac{1}{2 m} \sum_{i=1}^{m}\left(h_{\theta}\left(x^{(i)}\right)-y^{(i)}\right)^{2}
$$
公式解释：

先以简单的线性回归为例，假设公式为：
$$
h_{\theta}\left(x^{(i)}\right)=\theta_{0}+\theta_{1} x
$$
为了避免正负相抵消，求和号内预测结果与实际结果之差需要先求平方再相加；因为共有m个数据，故求和结果需要乘以1/m；而为了后续Gradient decent中对上式求导后所得公式的简便性，将公式再乘以1/2（也就是没啥实际意义，为了好看）。

Cost function 表征了预测与实际的偏差程度，那么最好的结果当然是通过修改我们的预测来使这个 J 值达到最小，于是如何修改就变成了一个问题，我们先用两种简单的图示来表示一下这个过程：

1、J对于两个参数的分别的变化曲线（以x前的参数为例）

做出J在只改变某一参数的过程中的变化曲线，并选取使得 J 最小的参数。

2、等高线图

在等高线图中，每种颜色的线对应一个 J 的值，这种表示方法可以在一个图中同时反映出两个参数对于预测结果准确度的影响，并且可以直观的看到使 J 最小的坐标点。
