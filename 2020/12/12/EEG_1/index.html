What is EEG?

脑电图（Electroencephalogram,EEG）是通过精密的电子仪器，从头皮上将脑部自发性的生物电位加以放大记录而获得的图形，是通过电极记录下来的脑细胞群的自发性、节律性电活动。（记录装置有侵入型、半侵入型和非侵入型之分）

所谓自发自发性生物电位就是指动作电位（Action Potentials），是由于外界刺激或体内细胞活动导致产生的电信号，这里就不做解释的将兴奋性突触后电位（excitatory post-synaptic potential，EPSP）的细节图放在这里，如图1：

图1

与EPSP相互对应的还有抑制性突触后电位（inhibitory post-synaptic potential，IPSP），这两种电位在连续时间上的交替出现构成了我们所测量到的电位变化图；从人体内一个细胞的角度来看，通过细胞内微电极记录的神经元膜电位变化和电流流动，我们很容易通过推广理解一个动作电位的变化及传递过程，图2为前一个细胞产生EPSP或IPSP时该细胞产生电位及释放递质的示意:

图2

通过简单的描述，我们大概清楚了动作电位的产生以及传递的过程；我们要分析的是脑电，那么既然知道了是有电位产生的，我们接下来的任务就是记录并分析了。

记录EEG时电极的常见分布

在检测脑电信号时，我们最常采用的是10-20系统电极放置法。

10-20系统电极放置法是国际脑电图学会规定的标准电极放置法。额极中点至鼻根的距离和枕点至枕外粗隆的距离各占此连线全长的10％，其余各点均以此连线全长的20％相隔，如图3所示。因此命名为10-20系统。

图3

先不说为什么叫10-20系统，我们先来理解一下其中字母数字代表的含义：

分类及含义
数字 左脑单数，右脑双数；
“/“代表中间（C3/T3之两点之间的中间点）
字母 F：额叶（Frontal lobe）
Fp：前额叶（Frontal poles）
T：颞叶（Temporal lobes）
O：枕叶（Occipital lobes）
P：顶叶（Parietal lobes）
C：中心部（Central）
Z：零点（Zero）

详细来说，其排布服从以下规则：

10-20系统的电极位置主要以颅骨为参照，并不因个人头围或头型的差异而有所不同。10-20系统电极位置描述如下：

  1. 前后矢状线：从鼻根至枕外粗隆取一连线，在此线上，由前至后标出5个点，依次命名为：额极中点（Fpz）、额中点（Fz）、中央点（Cz）、顶点（Pz）、枕点（Oz）。额极中点至鼻根的距离和枕点至枕外粗隆的距离各占此连线全长的10％，其余各点均以此连线全长的20％相隔。
  2. 横位：从左耳前点（耳屏前颧弓根凹陷处）通过中央点至右耳前点取一连线，在此连线的左右两侧对称标出左颞中（T3）、右颞中（T4）、左中央（C3）、右中央（C4）。T3、T4点与耳前点的距离各占此线全长的10％，其余各点（包括Cz点）均以此连线全长的20％相隔。
  3. 侧位：从Fpz点向后通过T3、T4点至枕点分别取左右侧连线，在左右侧连线上由前至后对称地标出左额极（Fp1 ）、右额极（Fp2 ）、左前颞（F7）、右前颞（F8）、左后颞（T5）、右后颞（T6）、左枕（O1）、右枕（O2）各点。Fp1 、Fp2点至额极中点（Fpz ）的距离与O1、O2点至Oz点的距离各占此连线全长的10％，其余各点（包括T3、T4）均以此连线全长的20％相隔。
  4. 其余的左额（F3）、右额（F4）点分别位于Fp1、Fp2与C3、C4点的中间；左顶（P3）、右顶（P4）点分别位于C3、C4与O1、O2点的中间。

除了国际标准的10-20系统以外，还有：

  • 10-10系统：在10-20系统提出修正和增补，共有81个电极点，包含之前的21个电极点，每个电极相距10%的距离
  • 10-5系统：为了进一步提高空间分辨率而设计出的，使用128个甚至更多的电极进行记录
  • 其他方法，例如Gibbs法、Arid法、Gastaut法、Bickford法等，但是多用于医院临床和基础脑电研究，日常使用的便利性较差

多种电极的可参考电极放置文献

那么我们记录到了我们头皮上多个点位的电位，下一步就是考虑如何将其呈现为一种有特征的形式以便于进行分析了。

Montages

记录单独一个电极的电位变化情况是没有意义的，因为这样无法突出不同脑区之间的差异性，也就无法对整体其进行分析。在脑电分析中，我们通常使用差分放大器测量两个不同信号的电压之差，并将此结果作为一个EEG活动的通道（channel）。上述过程被称为Montages，

国际临床神经生理学会IFCN对于Montage的英文定义为：

The arrangement or array of channels on the EEG machine display, defined by the exploring and reference electrodes.

即电极的导联组合。我们通常使用的Montages主要有以下几种：

Bipolar Montages

Bipolar Montage记录的是头皮上两个不同电极信号之间的差异。Bipolar Montage通常比较的是某一条电极连线上每对相邻电极之间的电位差，根据选取连线的方式不同，可以分为以下三种：

类型 特点
bipolar montage 选取从额叶到枕叶（即从前到后）的电极连线,并将连线上的电极逐个对比，示意图及产生的对比结果如下图：
transverse bipolar montage 选取从左边到右边的电极连线（例如从左耳到右耳，或Fp1-Fz-Fp2连线均可），并将连线上的电极逐个对比，示意图及产生的对比结果如下图：
triangular bipolai montage 一组由三个呈三角形放置的、互相距离相等的电极成对组配成的导联。用于脑瘤患者，在头皮电极组成的等边三角形导联根据尖波、棘波、慢波或综合波所在导联进行定位，故又称“三角定位法”。该方法仅为对病理波部位进行定侧、定位的方法之一，只是在已发现异常时，增加一种显示的方式，本身不具有活化作用。

本种方法不使用参考电极，只选取两个位于头皮上的电极，这样可以大大减少干扰，并可能排除无关电极引起的误差。

Common Reference Montage

在使用Common Reference Montage的方法下，每次差分放大器记录的为头皮电极与参考电极之间的差异，在实验中，我们经常使用Cz或者A1、A2作为参考电极，该种方法的示意图及输出样例如图6所示：

Common Average Reference Montage

在采用本种方法时，我们会将一个电极的电位与其他所有电极电位的均值进行比较，如图七所示，我们在计算F7的Common Average Reference Montage时，会比较F7的电位和除F7外所有电极电位的均值：

但是，由于Fp1和Fp2容易出现眼动伪迹（主要来自于眼球的移动以及眨眼），以及O1和O2容易出现头动伪迹（head movement artifact），我们一般将其从用于取均值的电极总体中除去，得到结果如图8所示：

图8

但是在使用Common Average Reference Montage时必然有出现reference contamination的可能性，即如图9右侧所示，如果C3、Cz、C4处的负值非常大，就会拉低均值整体的大小，导致如图9左侧所示的，所有信号对于均值都有一个相对的正电压，导致本次记录的数据无效。

为了解决本方法的固有问题，就开发出了Laplacian Montage。

Laplacian Montage

在最简单的定义情况下，本种方式是将一个信号与其距离最近的信号的平均值相比较，如图10所示，T7与F7-C3-P7的均值进行比较：

图10

在存在reference contamination的情况中，本种方法就能很好的体现出其自身的作用，如图11右侧所示。在2处每个电极信号都出现了同样的波形，这样通过Common Average Reference是无法将eference contamination消除的，也就无法队这个时间段的信号进行分析，而如果使用Laplacian方法，就可以如图11右所示的得到一个平坦的图像，仅凭肉眼就可以看出此电位是由F8处产生的，极大的方便了后续的分析。

图11

但是Laplacian Montage对于广泛分布的异常信号的处理能力较差，有些时候无法帮我我们准确地找到出现异常的电极位置，此时可以使用一个距离异常电极区域较远的电极作为参考电极，这样一般情况下可以获得的较好的效果。

但我们通常不会在EEG的分析中使用本种方法，主要是因为本方法没有办法被很好地定义，无法产生一套普遍的标准。
